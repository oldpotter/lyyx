# 第十四讲：当代人为何难以悟道？

参禅的目的，在明心见性，就是要去掉自心的污染，实见自性的面目。

污染就是妄想执着，自性就是如来智慧德相。

如来智慧德相，为诸众生所同具，无二无别。

若离了妄想执着，就证得了自己的如来智慧德相，就是佛。

否则就是众生。

**经文：**

**阿难 。汝犹未明一切浮尘 。诸幻化相 。当处出生 。随处灭尽。幻妄称相 。其性真为妙觉明体 。如是乃至五阴六入 。从十二处 。至十八界 。因缘和合 。虚妄有生 。因缘别离 。虚妄名灭 。殊不知生灭去来本如来藏 。常住妙明。不动周圆妙真如性 。性真常中 。求于去来迷悟生死 。了无所得 。**
